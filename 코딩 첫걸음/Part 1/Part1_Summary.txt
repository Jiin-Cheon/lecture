Native App 과 Web App
Native App -> 따로따로
Web App -> IOS, Android 둘 다 한번에

적응형과 반응형
pc와 모바일을 사이트를 따로따로 만들기->적응형 웹 (무거운 페이지)
pc와 모바일 사이트를 같이 쓰기 -> 반응형 웹 (가벼운 페이지)

쿠키, 세션, 캐시
쿠키-> 특정 정보가 사용자의 브라우저에 저장하는 것 (사용자의 컴퓨터 저장)
세션-> 서버측에 특정 정보를 기록
캐시-> 페이지 로드 시간을 줄이기 위해 미리 다운로드 해놓는 파일들
